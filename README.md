Investment Behavior Dashboard

An interactive data analytics dashboard built to explore investment behavior and sentiment across 40 survey respondents.

Live Demo
https://divyakauluri-spec.github.io/investment-dashboard

Overview
This dashboard presents findings from a structured investment behavior survey, covering:
- Gender distribution and demographic overview
- Preferred investment avenues (Equity, Mutual Funds, Fixed Deposits, etc.)
- Reasons behind each investment choice
- Savings objectives and information sources
- Investment duration, expected returns, and age correlation

Built With
- HTML, CSS, JavaScript
- Plotly.js — interactive charts
- Google Fonts — Fraunces + IBM Plex Sans + IBM Plex Mono

How to Run Locally
No installation needed. Just download the three files and open index.html in your browser:
- index.html
- data.js
- app.js

Dataset
40 respondents, 24 variables — collected via structured survey instrument covering demographics, investment preferences, risk appetite, and return expectations.
