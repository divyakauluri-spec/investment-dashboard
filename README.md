Investment Dashboard

Overview

An interactive business analytics dashboard that visualizes investment performance and financial insights. The dashboard enables users to monitor trends, compare investment categories, and support better financial decision-making.

Problem Statement

Investors need clear visual insights to evaluate financial performance. This project transforms raw investment data into meaningful dashboards for analysis and decision-making.

Features

- Investment Overview
- Performance Analysis
- Interactive Filters
- KPI Cards
- Financial Visualizations
- Trend Analysis

Technologies Used
